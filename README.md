# Vista Social Bulk Schedule — April 2026 Brand Content

Public media host for Qubit Capital brand content scheduled via Vista Social bulk CSV import.

## Structure

- `carousels/C1/ … C10/` — 5 slides per carousel (`slide_01.png` … `slide_05.png`)
- `statics/` — 10 static image posts

## Usage

Raw URLs follow this pattern:
`https://raw.githubusercontent.com/saurabhbaby/vista-social-bulk-schedule/main/<path>`

These URLs are referenced in `schedule.csv` for bulk upload to Vista Social.
